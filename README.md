# Exonviz
Visualise exons
------------------------------------------------------------------------
## Usage
Pass a transcript to exonviz to generate a figure.

## Examples
```bash
exonviz NM_004006.3:c.= > dmd.svg
exonviz NM_000533.5:c.= > plp1.svg
exonviz ENST00000358273.9:c.= >nf1-202.svg
```
